# Code Of Conduct

## Our Commitment

This project aims to provide a respectful, constructive, and technically serious environment for contributors and users.

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

- being respectful and professional
- assuming good intent while giving direct technical feedback
- focusing discussion on the work rather than the person
- documenting concerns clearly, especially around safety and correctness
- accepting that trading-related software requires careful review and caution

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, intimidation, or personal attacks
- abusive, insulting, or discriminatory language
- bad-faith disruption of discussions or reviews
- publishing private, security-sensitive, or credential-related information without permission
- encouraging unsafe or deceptive use of the software

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that do not align with this Code of Conduct.

## Scope

This Code of Conduct applies within project spaces and in public interactions representing the project.

## Reporting

If you experience or witness unacceptable behavior, report it to the repository owner through GitHub.
